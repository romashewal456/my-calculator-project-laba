# Простий Калькулятор на Python

Цей репозиторій містить простий консольний додаток-калькулятор, написаний на Python, та повний набір юніт-тестів для перевірки його функціональності.

## Функціонал

Модуль `calculator.py` реалізує такі операції:

* Додавання: `add(a, b)`
* Віднімання: `subtract(a, b)`
* Множення: `multiply(a, b)`
* Ділення: `divide(a, b)` (з обробкою винятку ділення на нуль).

---

## Запуск Тестів

Для перевірки коректності роботи всіх функцій модуля використовується стандартна бібліотека **`unittest`**.

1.  **Перейдіть до директорії проєкту:**
    ```bash
    cd my-calculator-project
    ```

2.  **Запустіть тести:**
    ```bash
    python -m unittest test_calculator.py
    ```

**Очікуваний результат:** Усі тести повинні пройти зі статусом `OK`.
